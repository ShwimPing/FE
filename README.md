# 🛖쉼핑
통합 공공 쉼터 및 대피소 정보 제공 플랫폼
<br/>
<br/>


## ✨주요 기능
- 지도에서 카테고리별 쉼터 위치 확인
- 음성 인식을 통한 장소 검색
- 장소별 상세 내용 확인(리뷰, 휴관일, 북마크)
- 리뷰 작성
- 폭염과 한파 등에 관한 카드뉴스 제공
- 푸시 알림을 통해 폭염 또는 한파 시 사용자에게 알림 전송
<br/>
